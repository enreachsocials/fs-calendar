# Future Schooling · Plan Contentu

Interaktywny kalendarz contentu dla klientki — 13.07–16.08.2026.

## Jak zaktualizować

1. Edytuj `index.html`
2. Wypchnij zmiany do GitHub (`git push`)
3. Vercel automatycznie wdroży nową wersję

## Funkcje

- Nawigacja tydzień po tygodniu (strzałki lub klawiatura ← →)
- Karty postów: Story, Image, Reel, Video, Kampania, Nagrywka
- Notatki Eli przy każdym poście (zapisują się automatycznie)
- Sekcja "Ela — pomysły na content" na dole każdego tygodnia
- Tryb jasny / ciemny
- Responsywny — działa na desktop i mobile
